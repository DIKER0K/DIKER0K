Напиши код бота ВК на Pythob
  Конечный код бота ВК на Python:  import vk_api from vk_api.longpoll import VkLongPoll, VkEventType  vk_session = vk_api.VkApi(token='токен_группы') longpoll = VkLongPoll(vk_session) vk = vk_session.get_api()  for event in longpoll.listen():     if event.type == VkEventType.MESSAGE_NEW and event.to_me:         message = event.text.lower()         user_id = event.user_id                  if message == 'привет':             vk.messages.send(user_id=user_id, message='Привет!')         elif message == 'как дела?':             vk.messages.send(user_id=user_id, message='Хорошо, а у тебя?')         elif 'мем' in message:             vk.messages.send(user_id=user_id, attachment='photo-12345678_901234567')         else:             vk.messages.send(user_id=user_id, message='Не понимаю, что ты хочешь.')   В этом коде бот просто отвечает на сообщения пользователя, используя методы VK API. Он может реагировать на ключевые слова и отправлять фото с помощью функции вложения.
